Link's Crystal Game

Instructions:
  • You will be given a random number at the start of the game.
  • There are four gems, clicking them will add a specific ammount to your current score.
  • You win the game by matching your score to the target number, and lose by going over.
  • The value of each gem is hidden, you will discover it by clicking on it.
  • Each time the game starts the values of the gems will change.
